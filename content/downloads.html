The latest stable release of Apache Traffic Server is v9.1.3, released on 2022-08-09. In addition, we continue to support the v8.1.x LTS release train, currently v8.1.5, which was released on 2022-08-09. We follow the Semantic Versioning scheme. The goal is to release patch releases frequently, and minor releases as needed. Within the major versions, all such patch and minor releases are all compatible.

When downloading from a mirror, please be sure to verify the hash and OpenPGP compatible signature. To validate the PGP signatures the public keys from the release manager will be needed. You can also browse through all releases and hash signatures.
